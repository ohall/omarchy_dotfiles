# Omarchy Dotfiles

Personal dotfiles for Omarchy/Hyprland setup on Arch Linux.

## Contents

- `.config/` - Application configs
  - `hypr/` - Hyprland window manager
  - `waybar/` - Status bar
  - `walker/` - Application launcher
  - `alacritty/`, `kitty/`, `ghostty/` - Terminal emulators
  - `nvim/` - Neovim
  - `mako/` - Notifications
  - And more...
- `.bashrc` - Shell configuration
- `.XCompose` - Custom compose key sequences

## Installation

```bash
# Clone
git clone git@github.com:ohall/omarchy_dotfiles.git

# Symlink configs (example)
ln -sf ~/omarchy_dotfiles/.config/hypr ~/.config/hypr
```
